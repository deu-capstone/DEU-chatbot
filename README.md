# 동의대학교 학사 공지사항 RAG 챗봇 시스템
동의대학교 학생들을 위한 학사 상담 및 공지사항 추천 RAG(Retrieval-Augmented Generation) 챗봇 시스템입니다. 학교 홈페이지의 주요 공지사항을 수집하여, 학생들의 질문에 정확하고 출처가 명확한 답변을 제공합니다.

## 주요기능
### 동의대 공지사항 크롤링
  #### 1. 과거 데이터 구축
  - 수집 기준: 2026년 2월 1일 이후로 작성된 게시글만 필터링하여 수집
  - 수집 범위: 동의대학교의 일반공지, 장학공지, 학사공지, 취업공지, 교육•모집공지, 채용공지로 총 6개의 카테고리를 수집
  - 본문 이미지를 로컬에 다운로드하고 본문 내용과 본문 이미지의 경로와 제목은 deu_notices.json으로 저장

  #### 2. 본문 이미지 텍스트 추출
  - 크롤링된 공지사항 메타 데이터를 바탕으로 다운로드 된 본문 이미지의 텍스트와 본문 내용을 병합하여 deu_notices_parsed.json으로 저장
  - Tesseract OCR 을 사용하여 이미지 파일 텍스트를 추출하고 정규표현식을 통해 노이즈를 필터링

  #### 3. 신규 데이터 수집
  - 게시판별 최신 글 번호가 deu_update_status.json에 기록
  - deu_update_status.json을 통해 새로운 게시글만 수집

###  RAG 기반 동의대 챗봇 기능 구현
  #### 1. 벡터 데이터베이스 구축 및 관리
  - 파싱된 JSON 데이터를 LangChain의 RecursiveCharacterTextSplitte를 사용하여 분할
  - OpenAI 임베딩 모델을 활용하여 문서를 벡터화하고 로컬 Chroma DB에 저장
  - 데이터 갱신 시 고유 ID 기반의 Upsert 로직을 적용하여 데이터 중복을 방지하고 빠르고 안정적인 증분 업데이트 지원

  #### 2. 출처 기반 질의응답 체인
  - 사용자 질문에 대해 벡터 DB에서 의미상 가장 관련성 높은 문서 조각을 검색
  - gpt-4o-mini 모델을 활용하여 검색된 참고 문서를 바탕으로 답변 생성
  - 프롬프트 엔지니어링을 통해 답변 내에 [공지사항 제목], [작성일], [출처 링크]를 반드시 명시하도록 하여 환각 현상을 차단

  #### 3. 문맥 기반 맞춤형 추천 시스템
  - 사용자의 소속 학과나 이전 검색 기록으로 쿼리 생성
  - 학과 정보가 담긴 쿼리와 검색 기록이 담긴 쿼리를 이용하여 벡터 유사도 검색을 통해 학과 맞춤 공지사항과 학생 관심사 맞춤 공지사항을 각각 5개까지 중복 없이 필터링하여 제공

  #### 4. 업데이트 파이프라인
  - run_pipeline.py 스크립트를 통해 [최신 글 크롤링 -> 이미지 파싱 -> 벡터 DB 업데이트 API 호출]을 한 번에 처리
